Q&A

Q1 : What are the three main differences between web 1.0, web 2.0 and web 3.0 ?

1. Advertising was executed differently, in web 1.0 there was only Banner Advertising
while web 2.0 introduced the world Interactive Advertising letting the user take action.
In web 3.0 there is Behavioural Advertising using information gathering.
2. Web 1.0 was able to preview read-only static web, web 2.0 started using read-write wildly and was more web interactive.
Web 3.0 was all read-write enabling high intelligence web.
3. Each one of the web version was focused on other population. web 1.0 was focused on the business side and was designated to companies.
web 2.0 was designed to communities while web 3.0 was focused on individuals.

Q2 : How would you search "drugs side-effects" ?

Web 1.0 : understanding difference number 3 in order to search for a question I would probably go to a drugs company
and search there manually as search engines wasnt efficient. Web 2.0 : As search engines was getting better I could use such one as Google and create a generic query "drugs side-effects"
and search for the most relevant for me. Web 3.0 : In web 3.0 I can search the specific drug/information I would like to know.
